ステータス表示 Status display

3色ですべてがわかる Three colors tell you everything

複数セッションが同時に動いていても大丈夫。最も緊急度の高い状態がメニューバーに表示されます。 Running multiple sessions? No problem — the most urgent state always wins in the menu bar.

待機中Idle

タスクが完了し、次の指示を待っている状態。安心して目を離せます。 The task is done and the agent is waiting for your next instruction. Safe to look away.

作業中Working

エージェントがコードを読み書き・実行している最中。まだ手を出さなくて大丈夫。 The agent is actively reading, writing, or running code. Nothing to do yet.

0:07

承認待ちNeeds approval

あなたの操作が必要な状態。点滅しながら経過時間を表示し、放置を防ぎます。 Your action is required. It blinks and counts elapsed time so it never gets forgotten.

特徴 Features

静かに、確実に知らせる Quiet, but never missed

🔇

音なし・ポップアップなしNo sound, no popups

通知音もダイアログも出しません。Web会議中でも他人に気づかれずに状態を確認できます。 No alert sounds, no dialogs. Check status discreetly, even mid-video-call.

🚥

最も緊急な状態が勝つMost urgent state wins

複数のエージェントを並走させても、メニューバーは常に一番対応が必要な色を表示します。 Run several agents at once — the menu bar always shows the color that needs you most.

📱

ntfyでスマホにも通知Push to your phone via ntfy

離席中の承認待ちは、ntfy連携でスマホへプッシュ通知。任意設定・デフォルトはOFF。 Optional ntfy integration pushes "needs approval" alerts to your phone while you're away. Off by default.

🏭

由来はトヨタ生産方式Named after the Toyota andon

生産ラインが止まったときに点灯する「アンドン(行灯)」から着想。異常を人に伝える現場の知恵をそのまま。 Named after the andon lamp that lights up when a production line needs human attention — same idea, for your terminal.
